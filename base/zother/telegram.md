#### 注册

方式一: 购买现成账号

```sh
1.准备科学上网
2.sms activate官网
	#注册账号,充值
	#购买现成账户 
		a.选目标teltegram数据 (查看内容包括:tdata json + session) 
		b.点购买,(此处可能需要账户验证,去注册邮箱获取验证码)
3.下载购买内容 xxx.zip,解压到某目录,比如 c:/tele
4.telegram官网下载telegram便携版
	#官网 -> tele for pc/linux -> portable version
	#解压到上述购买内容解压目录的tdata同级目录
	#解压后,有一个Telegram目录,将里面的Telegram.exe程序剪切到tdata目录同级
5.打开Telegram.exe,即可正常使用(无需登录)
6.汉化
	#telegram界面搜索zh_cn,选择中文语言包,安装即可
7,改密:
	#telegram设置->隐私与安全->两步验证
	#输入旧密码:在tdata目录同级的2FA.txt里面		//不同版本可能有所差异
	#输入新密码,找回邮箱(我的时gmail邮箱)


# 多个账号,上述重复,多个Telegramm.exe
```

方式二: 购买短信验证

```sh
1.科学上网
2.在sms activate官网购买telegram短信(最好英国,成功率高点)
3.打开购买的虚拟号,等待接收短信 (如2分钟内未收到则取消,时限内取消不扣款)
4.下载telegramx,注册:输入购买的虚拟号码,输入接收的短信,...操作成功
5.下载pc端telegram,注册:输入购买的虚拟号,会向telegramx发送验证码,输入收到的验证码即可
```
