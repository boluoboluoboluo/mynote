## Surfshark

### 1. 注册 & 购买账号（最重要一步）

1. 打开官网：[https://surfshark.com](https://surfshark.com/?referrer=grok.com)
2. 点击右上角 **Get Surfshark** 或 **Pricing**
3. 选择套餐（推荐 24/36 个月长期套餐，月均最便宜，通常有 80–87% 折扣 + 额外几个月免费）
   - **Starter**：纯 VPN（最便宜）
   - **One**：VPN + 杀毒 + 数据泄露监控（性价比高）
   - **One+**：再加私人搜索 + 假身份信息（Alternative ID）
4. 用邮箱注册（建议用非敏感邮箱），支付（支持支付宝/微信/银联/信用卡/PayPal/加密货币等）
5. 支付成功后，立刻收到登录凭证（邮箱 + 密码）。**记住：Surfshark 用的是“邮箱+密码”登录，不是用户名。**

### 2. 下载 & 安装 App（所有平台通用流程）

1. 登录官网账户 → 点 **My account** → **VPN** → **Manual setup** 或直接点 **Download apps**
2. 根据你的设备选择对应版本下载：
   - Windows / macOS：官网直接下载 .exe / .dmg
   - Android：Google Play 或官网 APK（墙内建议 APK）
   - iOS：App Store 搜索 “Surfshark”
   - Linux：官网有 .deb / .rpm 包 + 命令行 WireGuard / OpenVPN
   - 路由器 / Fire TV / Apple TV：见后面专用部分
3. 安装完成后打开 App → 输入刚才注册的**邮箱 + 密码**登录

### 3. 首次连接 & 日常使用（30秒上手）

1. 打开 App 后，默认在主界面
2. 点击大大的 **Quick-connect**（快速连接）按钮 → 自动选最快服务器（通常是香港、新加坡、日本、美国等）
3. 连接成功后顶部会显示 **Protected** / 绿色盾牌 + 当前 IP 所在地
4. 想换国家/地区：
   - 左侧 **Locations** → 搜索或滑动选择（有“最近使用”“最快”“静态IP”“P2P”分类）
   - 推荐日常用：**香港** / **日本-东京** / **新加坡** / **美国-洛杉矶/纽约**
5. 断开：再次点那个大按钮即可

### 4. 核心功能设置（强烈建议全部开启）

在 App 左侧菜单 → **Settings**（设置）里操作：

| 功能中文名     | 英文名                      | 推荐设置                 | 作用说明                                            |
| -------------- | --------------------------- | ------------------------ | --------------------------------------------------- |
| 自动连接       | Auto-connect                | 开启（Wi-Fi + 移动数据） | 每次连上不安全网络自动启动 VPN                      |
| Kill Switch    | Kill Switch                 | 开启（推荐 Advanced）    | VPN 掉线时自动断网，防止 IP 泄露（最重要安全功能）  |
| CleanWeb       | CleanWeb                    | 开启                     | 去广告 + 防恶意网站 + 防跟踪器（非常实用，省流量）  |
| NoBorders 模式 | NoBorders                   | 网络受限时开启           | 自动使用混淆技术，适合墙内/限制国家使用             |
| 协议选择       | Protocol                    | WireGuard（默认/最快）   | WireGuard > OpenVPN UDP > IKEv2 > OpenVPN TCP       |
| 多跳转         | MultiHop / Dynamic MultiHop | 按需开启（隐私最强时用） | 数据经过两个服务器，更难追踪（但速度会降20–40%）    |
| Bypasser       | Bypasser（分流）            | 按需设置                 | 某些 App/网站绕过 VPN（比如国内银行 App、微信支付） |
| Rotating IP    | Rotating IP                 | 可选开启                 | 每隔几分钟自动换 IP，适合极致隐私需求               |

**推荐新手安全组合**（全部开启）：

- Kill Switch → Advanced
- CleanWeb → On
- NoBorders → 墙内自动/手动开
- Protocol → WireGuard



### 常见问题 & 快速解决

| 问题                       | 解决办法                                                     |
| -------------------------- | ------------------------------------------------------------ |
| 连不上 / 很慢              | 换 WireGuard 协议 → 换 NoBorders 模式 → 手动选香港/日本/新加坡服务器 |
| 掉线频繁                   | 开启 Kill Switch → 关闭后台省电优化（安卓）                  |
| 国内某些 App 刷不出验证码  | 加 Bypasser，把支付宝/微信/银行 App 加入绕过列表             |
| Netflix / Disney+ 解锁失败 | 换美国/英国/日本等大服务器 → 开启 CleanWeb → 清浏览器缓存    |
| 想测速 / 确认 IP           | 连接后打开 https://surfshark.com/what-is-my-ip 或 ipleak.net |
