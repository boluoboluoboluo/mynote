##### 日语组成
```sh
平假名：长相圆滑，由草书变化而来
片假名：长相硬朗，由偏旁部首变化而来，记录外来语
汉字：由假名标注发音
罗马音：标注假名发音
```

```sh
假名包含：清音(50音)，浊音，半浊音，拗音，拨音
```

```
浊音：
	右上角加2个小点
	列： ka sa ta ha 	读：g z d b 
	词中词尾读浊音
半浊音：
	右上角加个小圈圈
	列：ha 	读：p
```

```
长音
	平假名长音，为第一排元音
	片假名长音，后面加一
短音

促音
	小号的(っッ)  #输入小号时,前加x
拗音
	除い以外的い段假名，和复元音や、ゆ、よ拼合起来的音节
送气音
	行：ka ta ha(浊音)
	位于词首发送气音
	位于词中词尾发不送气音
```
