





## Windows系统

### 方法1：控制面板卸载（推荐）

1. **打开控制面板** → **程序和功能**
2. 找到所有Python相关条目（可能有多个版本）
3. 右键选择“卸载”，按照提示操作

### 方法2：手动彻底删除

如果控制面板卸载不完整，需手动清理：

1. **删除安装目录**：

   cmd

   ```
   C:\PythonXX\ （XX为版本号）
   C:\Users\你的用户名\AppData\Local\Programs\Python\
   ```

   

2. **删除用户数据**：

   cmd

   ```
   C:\Users\你的用户名\AppData\Local\pip\
   C:\Users\你的用户名\AppData\Roaming\Python\
   C:\Users\你的用户名\AppData\Roaming\pip\
   ```

   

3. **清理环境变量**：

   - 右键“此电脑”→属性→高级系统设置→环境变量
   - 删除Path中的Python相关路径
   - 删除PYTHONPATH变量（如果存在）

4. **清理注册表**（谨慎操作）：

   - 按Win+R，输入`regedit`

   - 删除以下位置的相关键值：

     text

     ```
     HKEY_CURRENT_USER\Software\Python
     HKEY_LOCAL_MACHINE\SOFTWARE\Python
     ```



## Linux系统

### 方法1：APT包管理器（Ubuntu/Debian）

bash

```
# 1. 列出已安装的Python包
dpkg -l | grep python

# 2. 卸载Python
sudo apt purge python3 python3-pip python3-venv
sudo apt autoremove

# 3. 清理残留
sudo apt autoclean
```



### 方法2：源码编译安装的Python

bash

```
# 1. 找到安装位置
which python3

# 2. 进入源码目录（如果还有）
cd python-source-dir
sudo make uninstall

# 3. 手动删除
sudo rm -rf /usr/local/bin/python3*
sudo rm -rf /usr/local/lib/python3*
sudo rm -rf /usr/local/include/python3*
sudo rm -rf /usr/local/share/man/man1/python3*
```





## 通用清理步骤（所有系统）

### 1. 清理pip缓存

bash

```
# Windows
del /s /q %APPDATA%\pip\*
del /s /q %LOCALAPPDATA%\pip\*

# macOS/Linux
rm -rf ~/.cache/pip
rm -rf ~/.pip
```



### 2. 清理IDE配置

- **VS Code**：删除 `.vscode/extensions` 中的Python扩展
- **PyCharm**：删除 `~/.PyCharm*/` 或 `~/.config/JetBrains/` 相关配置

### 3. 验证卸载

bash

```
# 检查是否还有Python残留
python --version
python3 --version
pip --version
```
